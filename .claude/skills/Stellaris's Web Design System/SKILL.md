Quick reference:
- **Palette:** wood-brown `#5D4037` (primary, the only saturated hue), taupe `#D7CCC8`, cream bg `#FCF9F8`, nav beige `#ECE0DC`, ink `#2C2C2C`. No gradients, no blues/purples.
- **Type:** Hanken Grotesk (display/headlines), Manrope (body/UI, light 300 + line-height 2), Playfair Display Italic (wordmark only). All Google Fonts.
- **Feel:** modern-minimal, editorial whitespace, very square corners (buttons/images 2px), warm brown-tinted shadows, a signature full-width arrow rule under section heads.
- **Icons:** inline stroke SVG (Lucide/Feather match) for UI; Devicon for tech logos; emoji ❤️ for status only.
- **Copy:** Traditional Chinese, first-person, warm and earnest ("學習者", not "expert"); English overlines with `//` developer-comment prefix.
